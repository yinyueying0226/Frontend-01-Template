# 编程语言通识
## 语言按语法分类
1. 非形式语言
   1. 中文
   2. 英文
2. 形式语言
   1. 0型 无限制文法
   2. 1型 上下文相关文法
   3. 2型 上下文无关文法
   4. 3型 正则文法

## 产生式（BNF：巴克斯诺尔范式）
1. 用尖括号括起来的名称来表示语法结构名
2. 语法结构分成基础结构和需要用其他语法结构定义的复合结构
   1. 基础结构称终结符
   2. 复合结构称非终结符
3. 引号和中金啊的字符表示终结符
4. 可有有括号
5. * 表示重复多次
6. | 表示或
7. + 表示至少一次

### 通过生产式理解乔姆斯基谱系

## 其他生产式
    EBNF ABNF

## 现代语言的特性
1. C++ 中 * 可以表示乘号或者指针，具体是哪个取决于 * 钱的标识符是否被声明成类型
2. VB 中 < 可能是小于号，可能是 XML 直接量的开始，取决于当前位置是否可以接受 XML 直接量
3. Python 中行首的 tab 符和空格会根据上一行的首行空白以一定规则被处理成虚拟终结符 indent 或者 deduct
4. JS 中 / 可能是除号，也可能是正则表达式的开头，处理方式类似于 VB ，字符串模板中也需要特殊处理了，还有自动插入分号功能

## 图灵完备性
1. 命令式 图灵机
   1. goto
   2. if-while
2. 声明式 lambda
   1. 递归

## 静态与动态
1. 动态
   1. 在用户的设备或在线服务器上
   2. 产品实际运行时
   3. runtime
2. 静态
   1. 在程序员的设备上
   2. 产品开发时
   3. compiletime

## 系统类型 动态类型系统与静态类型系统

1. 强类型与弱类型
   1. string + number
   2. string == Boolean

2. 复合类型
   1. 结构体
   2. 函数签名

3. 子类型
   1. 逆变和协变
